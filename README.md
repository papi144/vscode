# VStudio v1.0 — Android Debug APK

A VS Code–styled code editor for Android, built with Jetpack Compose.

## Install

1. Tap the APK link below on your phone.
2. If Android blocks it: enable **Install unknown apps** for your browser, or open the file from a file manager.
3. Tap **Install**.

## Download

➡️ **[VStudio-v1.0-debug.apk](./VStudio-v1.0-debug.apk)** (~7.7 MB)

Min Android: 8.0 (API 26) · Target Android: 14 (API 34) · Package: `com.mavis.vscode`

## Features

- Activity bar (Explorer / Search / Source Control / Debug / Extensions)
- Tabbed editor with line numbers and monospace font
- ⌘K command palette (toggle terminal, save, save as, switch sidebar, etc.)
- In-app terminal/sandbox shell with `help`, `echo`, `date`, `pwd`, `ls`, `whoami`, `uname`, `clear`
- Save / Save-As writes to the app sandbox
- Dark theme matching the VS Code color palette

## Project source

The full Android project (Kotlin + Compose) is in `/workspace/vscode-android/` in the build environment.
